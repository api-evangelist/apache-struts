# Apache Struts (apache-struts)
Apache Struts is a free, open-source, MVC framework for creating elegant, modern Java web applications. It provides a clean separation between the model, view, and controller layers with a powerful convention-over-configuration approach.

## APIs

### Apache Struts
Struts provides a Java MVC framework API with action, interceptor, and result APIs, OGNL expression language for data binding, tag libraries for view rendering, and a REST plugin for building RESTful web services.
